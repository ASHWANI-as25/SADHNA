# ⚡ Quick Start Guide - SADHNA

## 🎯 Get Started in 3 Steps

### Step 1: Start Setup
- Click **"New Interview"** button in sidebar
- Or navigate to `/setup`

### Step 2: Configure Interview  
- **Select Role**: Software Engineer, Frontend, Backend, or HR Manager
- **Choose Difficulty**: Easy, Medium, or Hard
- **Upload Resume** (optional): PDF for HR analysis
- **Set Duration**: 10, 20, 30, 60, or 120 minutes
- **Enable Webcam** (optional): Click "Setup Webcam & Recording"

### Step 3: Start Interview
- Click **"Start Interview"** button
- Begin coding/answering questions
- Use hints if needed (Ctrl+H)
- Submit when ready (Ctrl+S)
- Skip if necessary (Ctrl+K)

---

## 💡 Essential Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` | Submit code |
| `Ctrl+K` | Skip question |
| `Ctrl+H` | Show/hide hints |
| `Ctrl+R` | Reset code |
| `?` | Show all shortcuts |

## ⚙️ Customize Experience

### Access Settings
- Click **Settings icon** (gear icon in top right)
- Or use menu in Layout component

### Available Settings
- **Theme**: Dark/Light mode
- **Font Size**: Small, Medium, Large
- **Sound**: Enable/Disable
- **Hints**: Show/Hide
- **Shortcuts**: Enable/Disable keyboard shortcuts
- **Recording**: Enable webcam recording
- **Transcription**: Enable real-time transcription

---

## 🎙️ Advanced Features

### Real-time Transcription
During interview, your spoken words appear as text
- Click microphone button to activate
- Transcription shows real-time as you speak
- Helps review what you said

### Webcam Recording
Before interview:
1. Click "Setup Webcam & Recording"
2. Allow camera/microphone access
3. Toggle record button to START recording
4. Recording indicator shows when active
5. Start interview - recording continues

### AI Feedback
After interview, receive feedback on:
- Overall score
- Strengths (what went well)
- Areas for improvement
- Technical assessment
- Communication evaluation

## 🆘 Troubleshooting

### "Microphone Not Working"
- Check browser permissions
- Allow microphone access when prompted
- Refresh page if needed

### "Webcam Permission Denied"
- Check browser settings
- Allow camera access
- Or continue without webcam

### "Code Not Running"
- Click "Run Tests" button
- Check syntax in code editor
- Review test cases in problem area
- Visit bottom console for errors

### "Settings Not Saved"
- Settings auto-save to browser localStorage
- Check if localStorage is enabled
- Try reset and save again
