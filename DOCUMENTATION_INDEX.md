## 📞 COMMON QUESTIONS

**Q: Where's the starfield code?**  
A: `src/components/AdvancedStarfield.jsx` (280 lines)

**Q: How do I use the animations in my own components?**  
A: Add CSS class: `className="animate-breathing-glow animate-floating-card"`

**Q: How to modify animation speed?**  
A: Override in CSS: `animation: floating-card 5s ease-in-out infinite;`

**Q: Is it mobile optimized?**  
A: Yes! Responsive at 480px, 768px, 1024px+ with reduced star count on mobile

**Q: What's the performance impact?**  
A: <3.6KB gzipped, <5% CPU, 60fps smooth

**Q: Can I test locally?**  
A: Yes! Run `npm run dev` and open browser to `http://localhost:5176`
